Документация на программу:

Приложение предназначено для получения метаданных о таблицах базы данных SQL Server и экспорта результатов в Word документ.

Функционал
	Ввод параметров подключения к БД
	Получение списка таблиц
	Просмотр столбцов выбранной таблицы
	Экспорт данных в Word файл

Структура 
	Клиентское приложение Windows Forms
	Библиотека классов DataBaseLibrary для взаимодействия с БД
	SQL Server для базы данных

Алгоритм работы
	Пользователь вводит параметры подключения к БД
	По нажатию кнопки выполняется подключение к БД
	Из БД запрашивается список таблиц, отображается в GridView
	При выборе таблицы запрашиваются метаданные столбцов
	По нажатию кнопки данные экспортируются в Word документ
