Документация на класс DataBase из пространства имен DataBaseLibrary

Класс DataBase предоставляет функциональность для работы с базой данных, включая получение списка таблиц и информации о столбцах таблиц. 
Класс обеспечивает возможность подключения к базе данных и выполнения запросов.

Структура класса:

- Namespace: DataBaseLibrary
  - Класс: DataBase
    - Поля:
      - string serverName - хранит имя сервера базы данных
      - string nameDB - хранит имя базы данных
      - string loginDB - хранит имя пользователя для подключения
      - string passwordDB - хранит пароль для подключения
      - string connectionString - строка подключения к базе данных
    - Метод: DatabaseTablesList() - получение списка таблиц базы данных
    - Метод: LoadColumnInformation(string tableName) - загрузка информации о структуре столбцов таблицы

Метод DatabaseTablesList():

	Этот метод выполняет подключение к базе данных и получает список таблиц.

	Параметры:
	Отсутствуют.

	Возвращаемое значение:
	DataTable - объект DataTable, содержащий список таблиц. Если произошла ошибка при подключении к базе данных, метод возвращает null.

Метод LoadColumnInformation(string tableName):

	Этот метод выполняет подключение к базе данных и загружает информацию о структуре столбцов выбранной таблицы.

	Параметры:
	- string tableName - имя таблицы, для которой нужно загрузить информацию о столбцах.

	Возвращаемое значение:
	DataTable - объект DataTable, содержащий информацию о структуре столбцов выбранной таблицы. 
	Если произошла ошибка при подключении к базе данных или выполнении запроса, метод возвращает null.
